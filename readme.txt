图片打包规则
1、文件重命名
如果是正方，文件名后加上-zf，反方则加上-ff， 建筑则加上-jz，gui则加上-ui，特效则加上-tx
或者文件名前、后加上正方、反方、建筑、特效
如文件夹dota1是反方，则需要重命名为“dota1-ff”,或是“ff-dota1”,或是“dota1反方”,或是“反方dota1”

2、svn地址需要放在 svn.txt格式为， 每个地址一行
如：
d:/dev/xx
g:/test/svn
3、code下的文件不能删，run.py，svn.txt不能删,不需要建立空目录、空文件夹,同时可以删除空目录、空文件夹

4、资源放在src目录下

5、python需要安装pillow库，使用python 3.5.2

6、安装kpjsfl
从 https://github.com/masterKPr/KPJSFL/下载jsfl库，运行source目录下的Installer.jsfl

7、使用方法
双击python2run.bat
